# Anfragen — Fracht, Paletten & Container (Deploy-Repo)

Öffentliches **Deploy-Repo** der Anfragen-App: Frachtanfragen verwalten,
Paletten und Container berechnen — offline-fähig, lokal-first, ohne Cloud.

**Live:** https://rawkeep.github.io/anfragen-app/

> Am Handy öffnen → „Zum Startbildschirm hinzufügen" — danach läuft die App
> als PWA auch offline. Alle Daten bleiben ausschließlich im Browser
> (localStorage); es werden keine Daten an einen Server übertragen.

Die Entwicklung findet in einem separaten (privaten) Repo statt; dieses Repo
spiegelt nur den auslieferbaren Stand der App. Jeder Push auf `main` wird
automatisch per GitHub Actions auf GitHub Pages veröffentlicht
(`.github/workflows/pages.yml`).
